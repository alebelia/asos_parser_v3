# Asos Parser

Программа для отслеживания цен на товары с asos.com/ru

Файл *asos_functions.py* содержит функции и переменные, необходимые для работы программы.
<br>Файл *asos_add_item.py* предназначен для добавления товаров в список отслеживания.
<br>Файл *asos_check_price.py* предназначен для проверки актуальной цены на товары, добавленные в список отслеживания, и оповещения по электронной почте в случае ее изменения.
